% Section 2
\section{\sectiontitleII}

\frame{
\frametitle{\sectiontitleII}

The exact value of a variable is called the \hspcu value. The value of the variables as indicated by a
measurement system is called the {\B \hspcu value}. The \hspcu of a measurement refers to the
closeness of agreement between the \hspcu value and the \hspcu value. But the {\G \hspcu value} is rarely
known {\it exactly}, and various influences, called \hspcu, have an effect on both of these values. So the
concept of the \hspcu of a measurement is a {\it qualitative} one. \vspcc

\begin{framed}\scalebox{1}{$\hspace{10mm}{\R error} = {\B measured\hspace{1mm}value} - {\G true\hspace{1mm}value} $}\end{framed}

\vspace{0mm}
{\tiny Text: Theory and Design of Mech. Meas.}
}

% Section 3
\section{\sectiontitleIII}

\frame{
\frametitle{\sectiontitleIII}

The {\bf \G true value} can be estimated but cannot be known  {\it exactly}. In practice a {\BR reference} value is used in place of the true value. We will discuss this again the the {\it Calibration Module}.


\begin{framed}\scalebox{1}{$\hspace{20mm} {\PR accuracy} =  \frac{|{\R error}|}{{\BR reference\hspace{1mm}value}}\times 100 $}\end{framed}

An estimate of error based using this value is sometimes referred to as {\B \hspcu accuracy}. \vspc

}

% Section 4
\section{\sectiontitleIV}

\frame{
\frametitle{\sectiontitleIV}

``The {\B \hspcu} is a numerical estimate of the possible range of the error in a measurement. In any
measurement, the {\R \hspcu} is not known exactly since the true value is rarely known exactly. But based on
available information, the operator might feel confident that the error is within certain bounds, a plus
or minus range of the indicated reading. This is the assigned \hspcu.''\vspace{5mm}\\
We will discuss this again the the {\it Uncertainty Module}.\vspace{10mm}\\

{\tiny Text: Theory and Design of Mech. Meas.}
}
